Для самостоятельного изготовления жидкостей для электронных сигарет необходимым компонентом является ароматизатор! Именно «аромка» придает основе для самозамеса предпочтительный вкус. Существующие «аромы» обладают как моно-вкусами, так и готовым сложными сочетаниями, например, вкусы ореховых пирожных, клубники со сливками и многих других.

Ароматизаторы можно использовать, сочетая различные вкусы между собой, добиваясь весьма интересных и необычных результатов. С помощью рецептов можно скопировать вашу любимую премиум-жидкость. Для самозамеса отлично подходят «аромки» американских марок TPA и Capella, итальянские FlavourArt, а также и отечественные ароматизаторы известной марки Smok Kitchen Jam.

Определить нужное количество ароматизатора можно с помощью рецептов в интернете, либо по вкусу, определяя нужное соотношение в практике. Для популярных вкусов TPA и Capella в интернете есть большое количество рецептов, где указано нужное количество на единицу объема основы. Опираясь на рецепты, вы сможете получить наилучшие результаты от самозамеса.

Внимание! Ароматизатор – это компонент для изготовления жидкостей, использовать ароматизатор в качестве заливки в электронную сигарету нельзя!
